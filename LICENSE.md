## 🔧 Troubleshooting & Common Issues

### ❌ Injection Fails Immediately

**Cause:** The target application may not be fully loaded yet, or injection depth is too shallow.

**Fix:**
```bash
lua-executor --inject-depth=4 --timeout=7000
```
Wait until the target application's main process is fully initialized before executing.

---

### ❌ APK Won't Install on Android

**Cause:** Unknown sources not enabled, or the APK is being blocked by Play Protect.

**Fix:** Go to **Settings → Biometrics & Security → Install Unknown Apps** and whitelist your file manager. If Play Protect flags the file, tap **Install Anyway** — this is common for sideloaded apps.

---

### ❌ macOS Says "App is Damaged"

**Cause:** Quarantine flag applied by Safari during download.

**Fix:**
```bash
sudo xattr -rd com.apple.quarantine /Applications/LuaExecutor.app
```

---

### ❌ Scripts Run But Produce No Output

**Cause:** Verbose mode or logging is disabled; output is being silently swallowed.

**Fix:**
```bash
lua-executor --verbose --log-output=~/Desktop/debug.txt
```

---

### ❌ Cloud Sync Not Working

**Cause:** API URL misconfigured or no internet connection during sync attempt.

**Fix:** Reset to default API URL:
```bash
lua-executor --api-url=default --verbose
```

---

### ❌ High RAM Usage After Long Sessions

**Cause:** Memory accumulation from large or looping scripts.

**Fix:** Enable aggressive garbage collection in config:
```bash
lua-executor --gc-mode=aggressive --gc-interval=30
```
